# ☕ EstudosJava

Bem-vindo ao meu repositório de estudos em Java! 🚀

Este espaço foi criado com o objetivo de documentar a minha jornada de aprendizado na linguagem Java, criando uma linha do tempo clara da minha evolução técnica — desde os fundamentos até o desenvolvimento de aplicações completas.

## 🎯 Objetivo

- **Consistência:** Manter uma rotina de prática diária, refletida no meu histórico de contribuições (streak) aqui no GitHub.
- **Portfólio:** Demonstrar para recrutadores e outros desenvolvedores a qualidade do meu código, organização e capacidade de aprendizado contínuo.
- **Preparação:** Construir uma base sólida em Java e Spring Boot para atuar como Desenvolvedor Júnior.

## 📚 Contexto e Cursos

Os códigos e projetos presentes neste repositório são, em sua maioria, desenvolvidos durante o **Curso de Java Completo (Nélio Alves - Udemy)**, além de desafios e práticas independentes.

## 🛠️ Tecnologias e Tópicos Abordados

No decorrer dos commits, você encontrará implementações envolvendo:

* **Fundamentos:** Lógica de programação, estruturas de controle e repetição.
* **Orientação a Objetos (POO):** Classes, atributos, métodos, encapsulamento, herança e polimorfismo.
* **Tratamento de Exceções:** Criação de exceções personalizadas e blocos try-catch.
* **Estruturas de Dados:** Arrays, Listas, Sets e Maps.
* **Tópicos Avançados:** Expressões Lambda, Generics, Interface e manipulação de arquivos.

## 📂 Como navegar neste repositório

Os arquivos estão organizados de forma cronológica ou por módulos de estudo.
